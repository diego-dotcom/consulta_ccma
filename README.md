# App de consulta CCMA
App de escritorio para consultar CCMA (ARCA) a través de API
  
### Puesta en marcha

1 - Descargue los archivos y colóquelos en una misma carpeta.  
2 - Ejecute app.exe.

### Obtención de API key

1 - En la app, seleccione la solapa "Obtener API key".  
2 - Ingrese su email y haga click sobre el botón "Obtener API key".  
3 - Recibirá la API key en el email indicado. Podrá procesar 10 archivos por mes calendario.  

### Modo de uso

1 - En la app, solapa "Procesar Archivo", complete con su email y su API key. Asegúrese de que no queden espacios en blanco si copia y pega.  
2 - Cargue un archivo .xls preparado por usted de acuerdo al ejemplo proporcionado, puede editar el archivo claves.xls descargado.  
3 - Haga click en "Procesar archivo" y espere. Por favor no cierre el programa.  
4 - Una vez finalizado el proceso, en la carpeta del programa se creará un archivo con la consulta de las CCMA, y otro con errores que pudiera hacer con algún CUIT.
